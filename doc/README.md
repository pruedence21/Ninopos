## Getting Started

1. Clone the repository
2. Copy `.env.example` to `.env.local`
3. Configure environment variables
4. Run `bun install`
5. Run `bun run db:push` to setup database
6. Run `bun run db:seed` to seed initial data
7. Run `bun run dev` to start development server
